## 3. ヨッシーアイランドUI調査

### ビジュアル特徴

- **手描きアウトライン:** 太さが不規則に変化するクレヨン風の輪郭線
- **テクスチャ:** 画用紙・クラフト紙の質感、水彩のにじみ
- **カラーパレット:** 彩度を落としたパステルトーン、純粋な黒は使わない
- **UIエレメント:** 波打った枠線、ぷにっと変形するボタン
- **フォント:** 丸みのある手書き風、文字サイズが微妙に不揃い

### CSS実装手法

**手描き風ボーダー（CSSのみ）:**
```css
.hand-drawn-box {
  border: 3px solid #4A4A4A;
  border-radius: 255px 15px 225px 15px / 15px 225px 15px 255px;
  background: #FFF8F0;
}
```

**画用紙テクスチャ:**
```css
.paper-bg {
  background-color: #FFF8F0;
  background-image: url("data:image/svg+xml,...feTurbulence...");
}
```

**手書き風フォント（Google Fonts）:**
- 日本語: Yomogi（よもぎ）、Zen Kurenaido、Klee One、Hachi Maru Pop
- 英語: Caveat、Patrick Hand、Architects Daughter

**クラフト風ボタン:**
```css
.craft-button {
  font-family: 'Zen Kurenaido', sans-serif;
  background: #FFECD2;
  color: #5D4037;
  border: 2.5px solid #8D6E63;
  border-radius: 255px 15px 225px 15px / 15px 225px 15px 255px;
  box-shadow: 3px 3px 0px #D7CCC8;
}
```

**マスキングテープ風装飾:**
```css
.masking-tape {
  background: rgba(168, 213, 186, 0.7);
  transform: rotate(-2deg);
}
```

### ビジネス信頼感とのバランス

- 見出し・ボタンのみ手書き風フォント、入力フォームや名刺本体は品のある書体
- パステルカラーは背景と装飾のみ、CTAボタンはテラコッタでしっかり目立たせる
- テクスチャは微細（opacity 0.05〜0.1）
- 手描き風の「崩し」はボーダーとフォントに限定、レイアウトは整列

### キャラクターコラボUI設計

- キャラクター配置スロット（角/ヘッダー/透かし）
- コラボバッジ（マスキングテープ風「コラボ限定」ラベル）
- 著作権表示エリア（全画面共通フッター）
- ちいかわ・すみっコのシンプルな線画はクラフト風UIと親和性が非常に高い
